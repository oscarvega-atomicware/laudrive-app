Laudrive Test App
=====================

App  a desarrollar para Laudrive.

## Instalacion

Para instalar sigue los siguientes pasos:

```bash
$ npm install -g ionic
```

Then run: 

```bash
$ bower install
$ npm install
$ ionic state restore
$ ionic serve
```
Antes de iniciar la app se debe inicializar la aplicaci√≥n BackEnd, en http://localhost:1337.
Para hacer esto simplemente vamos a la carpeta del proyecto backent y corremos el comando
```bash
$ sails lift
```
